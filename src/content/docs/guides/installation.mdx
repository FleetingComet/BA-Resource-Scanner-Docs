## Requirements

Before you begin, make sure you have the following:

- **Python** (v3.13 recommended; 3.8+ should work)
- **Virtual Environment** (highly recommended)
- **RapidOCR** - installed automatically via `requirements.txt`

<Aside type="tip">
  You do **not** need to install Tesseract or configure system paths. The scanner uses `RapidOCR`, a lightweight ONNX-based engine that runs entirely in Python.
</Aside>

## Install

<Steps>

1. **Get the project files:**

    <Tabs>
     <TabItem label="Option A - Clone with Git">
       ```bash
       git clone https://github.com/FleetingComet/Blue-Archive-Resource-Scanner.git
       ```
       ```bash
       cd Blue-Archive-Resource-Scanner
       ```
     </TabItem>
     <TabItem label="Option B - Download ZIP">
       1. Go to the repository page:
          [https://github.com/FleetingComet/Blue-Archive-Resource-Scanner](https://github.com/FleetingComet/Blue-Archive-Resource-Scanner)
       2. Click **Code -> Download ZIP**
       3. Extract the ZIP file to a folder
       4. Open a terminal inside the extracted folder
     </TabItem>
   </Tabs>


2. **Create and activate a virtual environment:**

   <Tabs syncKey="package-manager">
     <TabItem label="uv (recommended)">
       ```bash
       uv venv --python 3.13.0
       ```
       Then activate it:
       <Tabs syncKey="operating-system">
         <TabItem label="Windows">
           ```bash
           .venv\Scripts\activate
           ```
         </TabItem>
         <TabItem label="macOS / Linux">
           ```bash
           source .venv/bin/activate
           ```
         </TabItem>
       </Tabs>

     </TabItem>
     <TabItem label="pip / venv">
       ```bash
       python -m venv .venv
       # Windows
       .venv\Scripts\activate
       # macOS / Linux
       source .venv/bin/activate
       ```
     </TabItem>
   </Tabs>

3. **Install dependencies:**

   <Tabs syncKey="package-manager">
     <TabItem label="uv (recommended)">
       ```bash
       uv pip install -r requirements.txt
       ```
     </TabItem>
     <TabItem label="pip / venv">
       ```bash
       pip install -r requirements.txt
       ```
     </TabItem>
   </Tabs>

   _(This installs `rapidocr`, `onnxruntime`, `opencv-python`, and other required packages.)_

</Steps>

## Using UV for Dependency Management

This project uses [UV](https://github.com/astral-sh/uv) for streamlined Python package management via `pyproject.toml` and `uv.lock`. UV automates virtual environment setup, dependency resolution, and package installation.
