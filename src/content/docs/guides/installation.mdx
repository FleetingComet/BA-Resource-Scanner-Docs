## Requirements

Before you begin, make sure you have the following:

- **Python** (v3.13 recommended; 3.8+ should work)
- **Virtual Environment** (highly recommended)
- **RapidOCR** - installed automatically via `requirements.txt`

<Aside type="tip">
  You do **not** need to install Tesseract or configure system paths. The scanner uses `RapidOCR`, a lightweight ONNX-based engine that runs entirely in Python.
</Aside>

## Install

<Steps>

1. **Clone the repository:**

   ```bash
   git clone https://github.com/FleetingComet/BA-Scanner.git
   cd BA-Scanner
   ```

2. **Create and activate a virtual environment:**

   <Tabs>
     <TabItem label="uv (recommended)">
       ```bash
       uv venv --python 3.13.0
       ```
       Then activate it:
       ```bash
       # Windows
       .venv\Scripts\activate
       # macOS / Linux
       source .venv/bin/activate
       ```
     </TabItem>
     <TabItem label="pip / venv">
       ```bash
       python -m venv .venv
       # Windows
       .venv\Scripts\activate
       # macOS / Linux
       source .venv/bin/activate
       ```
     </TabItem>
   </Tabs>

3. **Install dependencies:**

   <Tabs>
     <TabItem label="uv">
       ```bash
       uv pip install -r requirements.txt
       ```
     </TabItem>
     <TabItem label="pip">
       ```bash
       pip install -r requirements.txt
       ```
     </TabItem>
   </Tabs>

   _(This installs `rapidocr`, `onnxruntime`, `opencv-python`, and other required packages.)_

</Steps>

## Using UV for Dependency Management

This project uses [UV](https://github.com/astral-sh/uv) for streamlined Python package management via `pyproject.toml` and `uv.lock`. UV automates virtual environment setup, dependency resolution, and package installation.
