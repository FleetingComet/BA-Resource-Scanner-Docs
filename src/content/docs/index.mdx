A Python-based tool to scan and count owned resources in **Blue Archive**.

<Aside type="caution">
  **Resolution**: Only supports **1280×720** resolution at this time.
</Aside>

## Features

<CardGrid>
  <Card title="Equipment & Items" icon="list-format">
    Automatically scan and count resources in the equipment and item page.
  </Card>
  <Card title="Student Page" icon="bars">
    Extract levels, skill tiers, gear, UE stars, and bond levels.
  </Card>
  <Card title="No External Binaries" icon="approve-check">
    Uses RapidOCR - a lightweight ONNX engine that runs entirely in Python. No Tesseract needed.
  </Card>
  <Card title="Data Sync" icon="down-caret">
    Optional online sync keeps your game asset data up to date automatically.
  </Card>
</CardGrid>
